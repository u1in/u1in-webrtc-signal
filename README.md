# u1in-webrtc-signal

一个集成了**基于 Node 信令服务器**和**配套客户端 js 库**的 Webrtc 信令服务工具

## 1. 安装

```bash
npm i
```

## 2. 构建

```bash
npm run build
```

## 3. 运行

将 output 文件夹复制到服务器上执行

```bash
npm i --production

node server.js
```

项目具备一个小的 demo 可以使用

```bash
npm run demo
```
